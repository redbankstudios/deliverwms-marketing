# Image assets

Drop hero and content images here. They'll be served from `/images/...` at runtime.

## Required

- **`hero-warehouse.jpg`** — homepage hero background. Landscape orientation, ideally 1920×1080 or larger. Used full-bleed with a dark gradient overlay on the left for text legibility.

## Recommended specs

- JPG or WebP, optimized to under ~300KB if possible (Squoosh.app is good for this)
- Aspect ratio: landscape, anything from 16:9 to 21:9 works
- Dimmer overall tone (or significant dark areas) so the white H1 + gradient overlay reads cleanly
